# Foodiegram

[Heroku link][heroku]

[heroku]: http://foodiegram.herokuapp.com/

Foodiegram is a clone of Instagram built on Rails and Backbone. Users can:

- Create accounts
- Create sessions (log in)
- View own posts
- View a feed of subscribed user posts
- Create posts
- Upload pictures
- Edit Posts and pictures
- User avatars
- Like/Unlike posts
- Comment on posts
- Subscribe to users
